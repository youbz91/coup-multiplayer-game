# Card Images Directory

Place your 5 role card images here with these **exact names**:

1. **duke.png** - Duke card image
2. **assassin.png** - Assassin card image
3. **captain.png** - Captain card image
4. **ambassador.png** - Ambassador card image
5. **contessa.png** - Contessa card image

Optional:
- **card-back.png** - Uniform card back for all cards (optional)

## File Requirements:
- Format: PNG (with transparency) or JPG
- Recommended size: 300x450px (portrait ratio 2:3)
- Max file size: 500KB per image

## After Adding Images:
The game will automatically display them with beautiful Hearthstone-style entrance animations!
